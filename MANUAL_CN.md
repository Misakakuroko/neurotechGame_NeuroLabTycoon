# NeuroLab Tycoon - 新手完全指南

欢迎来到 **NeuroLab Tycoon**！这是一款模拟核磁共振（MRI）实验室管理的硬核游戏。作为实验室主任，你的目标是通过设计和执行MRI扫描实验，积累资金（Money）和声望（Prestige），最终建成世界顶级的 **11.7T 实验室**。

这份指南将帮助零基础的你快速上手。

---

## 游戏核心流程

游戏分为三个主要阶段，循环进行：

1.  **采购与选题 (Lab Management)**：选择实验对象，购买硬件。
2.  **安全控制台 (Safety Console)**：调整扫描参数，确保实验安全且图像清晰。
3.  **结果评审 (Experiment Review)**：查看实验结果，获取奖励。

---

## 第一阶段：采购与选题 (Lab Management)

在这里，你需要决定今天要研究什么，以及用什么设备来研究。

### 1. 选择研究对象 (Research Target)
左侧面板是任务板。
*   **Phantom (模型)**：**新手必选**。无声望要求，难度最低。建议前几轮只做这个。
*   **Adult (成人)**：需要 20 声望。奖励正常。
*   **Pediatric (儿童)** / **Neonate (新生儿)**：高难度，高风险，高回报。对硬件和操作要求极高。

### 2. 购买硬件 (Hardware Shop)
右侧是商店。**注意：只有买齐 磁体 (Magnet)、冷却 (Cooling)、线圈 (Coil) 才能开始实验。**

*   **Main Magnet (磁体)**：核心部件。
    *   **3T**：入门级，便宜，安全。
    *   **7T / 11.7T**：高端设备，信号更强（**SNR更高**），但更难驾驭。
    *   *警告：升级到 7T 或 11.7T 前，必须确保有足够的钱买配套的高级冷却和 pTx 工具！*
*   **Cooling (冷却)**：
    *   **Standard (标准)**：仅适用于 3T。
    *   **Superfluid (超流体)**：**7T 和 11.7T 必须使用此冷却系统**，否则会发生"失超"（Quench，即磁体过热失效），导致实验直接失败。
*   **Gradients (梯度系统)**：
    *   决定了你能做多精细（Resolution）和多快（Sequence）的扫描。
    *   如果扫描分辨率高导致"过载"（Overload），你需要升级这里。
*   **Upgrades (升级)**：
    *   **pTx Tools**：**7T 和 11.7T 必须购买**，用于解决高场下的图像黑洞问题（B1+不均匀性）。

## 第二阶段：安全控制台 (Safety Console)

这是游戏最硬核的部分。你需要通过调整参数，让 **SNR (信噪比)** 尽可能高，同时保证 **Safety (安全)**。

### 核心指标（看右上角和中间）
*   **Predicted SNR (预测信噪比)**：**必须 > 20**。低于这个值图像全是噪点，实验失败。数值越高越好，决定了奖金倍率。
*   **Gradient Load (梯度负载)**：**必须 < 100%**。如果超标，请降低分辨率（数值调大）或去商店升级梯度系统。
*   **PNS Risk (神经刺激风险)**：**必须 < 100%**。过高会让受试者疼痛，导致实验中止。

### 参数调节指南
1.  **Pulse Sequence (序列)**：
    *   **GRE**：标准，平衡。
    *   **SE**：信号好，但慢。
    *   **EPI**：极快，但对硬件（梯度）压力巨大。容易导致设备过载。
2.  **Resolution (分辨率)**：
    *   数值越小（如 0.5mm）图像越清晰，但 **SNR 会成立方倍下降**（因为体素体积变小）。
    *   *新手建议：保持在 1.5mm - 2.0mm 左右。做 0.5mm 需要顶级的 11.7T 设备支持。*
3.  **Duration (扫描时长)**：
    *   时间越长，SNR 越高（图像越清楚）。
    *   *警告：如果超过 **10分钟**，受试者可能会动（Motion Artifact），导致实验有 50% 几率直接失败！新手建议控制在 10分钟以内。*
4.  **VOP Safety Model (N)**：
    *   这是安全计算模型的数量。
    *   **调大 N 值** = 安全计算更精确 = 允许使用更大功率 = **提高 SNR**。
    *   可以大胆调大，直到你觉得安全为止。
5.  **B0 Shimming (匀场) —— 必做操作！**
    *   看到左下角的 **Delta: xx Hz** 了吗？如果它是红色的（>60Hz），实验会失败。
    *   **玩法**：拖动 x, y, z 三个滑块，盯着那个数字，**让它越接近 0 越好**（变成绿色）。这模拟了真实的MRI匀场操作。
6.  **Checklist (核查表)**：
    *   右下角的三个勾选框（Physiological, Vestibular, Metallic Implants）。
    *   **必须全部勾选**，否则无法开始扫描。其中 **Metallic Implants**（金属植入物）是 MRI 绝对禁忌症，必须确认病人无金属残留。

## 第三阶段：结果评审 (Experiment Review)

点击 "Acquire Data" 后进入。你会看到控制台日志滚动。

*   **成功**：获得 Money（钱）和 Prestige（声望）。
*   **失败原因分析**：
    *   *Quench Detected* -> 你给高场磁体（7T/11.7T）配了低级冷却。
    *   *B1+ Inhomogeneity* -> 你用了高场磁体但没买 pTx 工具。
    *   *Shimming failed* -> 你没把匀场误差（Shimming）调到 60Hz 以下。
    *   *Gradient Failure* -> 梯度过载，去买更好的梯度或降低分辨率。
    *   *Motion Artifacts* -> 扫描时间太长（>10分钟），受试者动了。
